# Ethio Job Pro $ (Version 2.0) A global micro-tasking application with secure authentication and multi-channel payouts. ## New Features - **Dual Auth:** Support for Phone (OTP) and Email/Password login. - **Global Payouts:** Integrated fields for Telebirr, PayPal, and USDT. - **Security:** Automatic VPN detection and device fingerprinting. - **Referral System:** Automated bonus tracking for user invitations. ## Tech Stack - **Frontend:** Flutter / React Native - **Backend:** Firebase / Node.js - **Database:** Firestore / MongoDB
